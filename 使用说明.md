# 配电箱自动组价系统 - 使用说明

## 功能概述
本系统可自动从配电箱系统图中提取元器件信息，并根据ABB直通车、天工直通车、工控猫商城三个渠道进行询价，最终生成专业的Excel报价单。

## 系统要求
- Python 3.8+
- Windows/Linux/Mac均可运行

## 安装步骤

### 1. 安装Python（如果尚未安装）
下载地址：https://www.python.org/downloads/

### 2. 安装依赖
打开命令行/终端，进入项目目录，执行：

```bash
cd 配电箱自动组价系统
pip install -r requirements.txt
```

### 3. 启动系统
```bash
streamlit run app.py
```

系统启动后会自动打开浏览器，访问 http://localhost:8501

## 使用流程

### 第一步：上传配电箱系统图
1. 点击"选择文件"按钮
2. 选择配电箱系统图截图（支持PNG/JPG/BMP格式）
3. 支持一次性上传多张图片
4. 建议每张图片包含4-6个配电箱

### 第二步：设置品牌偏好
从下拉菜单选择元器件品牌：
- ABB + 施耐德（推荐）
- ABB
- 施耐德
- 优先ABB

### 第三步：识别并询价
点击"识别元器件并询价"按钮，系统将：
1. 自动识别图片中的元器件
2. 按优先级查询三个渠道的价格
3. 计算每个元器件的小计
4. 计算每个配电箱的总价

### 第四步：查看结果
- 查看每个配电箱的元器件清单
- 查看单价和询价来源
- 查看整体报价汇总

### 第五步：下载报价单
点击"下载Excel报价单"按钮，获取完整的Excel文件，包含：
- 报价汇总表
- 每个配电箱的详细报价表

## 询价渠道说明

| 优先级 | 渠道 | 说明 |
|--------|------|------|
| 1 | ABB直通车 | ABB官方商城，价格最权威 |
| 2 | 天工直通车 | 专业电气选型平台，覆盖全品牌 |
| 3 | 工控猫商城 | 施耐德官方授权，正品保障 |

## 注意事项

1. **图片质量**：上传清晰的配电箱系统图，确保元器件型号可见
2. **价格参考**：当前版本价格为模拟参考价，实际采购前请以网站实时价格为准
3. **多图处理**：建议将大型图纸拆分为多张截图，每张包含4-6个配电箱

## 常见问题

### Q: 启动报错"Module not found"
A: 请确保已执行 `pip install -r requirements.txt`

### Q: 图片识别不准确
A: 请上传更清晰的图片，确保元器件型号文字清晰可见

### Q: 无法获取实时价格
A: 当前版本使用模拟价格，实际使用时需要接入各平台的API或进行网页抓取
